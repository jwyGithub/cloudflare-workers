# vps-parse

## 环境变量配置

| 变量KEY         | 说明                  | 默认值 | 是否必填 |
| --------------- | --------------------- | ------ | -------- |
| GITHUB_TOKEN    | 推送到github时的token |        | 是       |
| GITHUB_USERNAME | github用户名          |        | 是       |
| REPO_NAME       | github仓库名          |        | 是       |
| REPO_BRANCH     | github仓库分支        |        | 是       |
| LINKS           | 需要被解析的订阅链接  |        | 否       |
| SUBS            | 转换的vps链接         |        | 否       |
| UPLOAD_URL      | 转换后上传的服务      |        | 否       |
| REMOTE_CONFIG   | 转换时的远端配置      |        | 否       |
| SUB_CONVERT     | 转换后端服务          |        | 否       |
